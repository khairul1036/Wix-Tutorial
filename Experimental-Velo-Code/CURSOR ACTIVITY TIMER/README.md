## 🚀 Installation Guide

### Step 1: Open Wix Dashboard
Go to:

Wix Dashboard → Settings → Advanced → Custom Code

### Step 2: Add Custom Code
- Click **Add Custom Code**
- Paste the provided JavaScript snippet
- Select **Body – End**
- Apply to **All Pages**

### Step 3: Publish
Publish your website to activate the script.

---

## 🧪 How to Test

1. Open your published site  
2. Right click → **Inspect**  
3. Open the **Browser Console**

You will see output like:

cursor moving: 1s  
cursor moving: 2s  
cursor stopped: 1s  
cursor stopped: 2s
